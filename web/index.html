SnoozeBot

SnoozeBot makes snoozing conversations in Help Scout super easy.

1) PICK A TIME
Click when you want to see this conversation again
2) SNOOZE
SnoozeBot adds a note and makes the conversation "Pending"
3) WAKE UP
When the time is up, SnoozeBot will automatically re-open your conversation and add a little note
It's just as easy to install

Step 1: Head to the Create Custom Appopen_in_new page in Help Scout

Step 2: Click Create Application

Step 3: Enter SnoozeBot as the App Name, set Content Type to be Dynamic and add a word to the Secret Key input box

Step 4: The Callback URL should be set to Snoozebot's App URL
https://www.snooze-bot.com/app
Step 5: Enable the Mailboxes you want SnoozeBot in and Save
SnoozeBot is in Beta Mode

SnoozeBot was designed and created by a Customer Support Team Lead who wanted to have a better way to stay on top of conversations throughout the day. SnoozeBot has been a weekend project and labor of love that I wanted to share with the amazing support community that uses and depends on Help Scout every day.

You should expect SnoozeBot to be around for awhile and work as described above, but it's probably not perfect. If you see any odd behavior or have some feedback about how the tool could be better, please do get in touch!

All the best on your Support Journey and happy Snoozing!
